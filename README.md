# Posts CRUD API
#### Rest API to implement CRUD for Posts using NodeJS, Express, Mongodb

## Features
### Posts
* Fetch all posts
* Fetch a post
* Create a post
* Delete post
* Update post
### Users
* Register
* Login

## Technology
* NodeJS
* Express
* Mongodb
* Mongoose ORM

## Installation
* Clone the app
* Install dependencies `npm i`
* Create and update `.env` file using `.env.example`
* Start the app. `npm start`
